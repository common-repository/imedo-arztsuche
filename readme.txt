Das Wordpress Plugin dient dazu, die imedo Arztsuche bequem ueber WordPress in die eigene Webseite zu integrieren.

== Description ==
**Das Arztsuche- Plugin: Alles auf einen Blick**

Die europaweite „e-Health Trend“-Studie ergab: Jeder dritte Deutsche informiert sich mindestens einmal pro Monat im Internet über Gesundheitsthemen.

Die Arztsuche von imedo bietet einen Überblick über das Angebot praktizierender Mediziner und unterstützt den Patienten mithilfe von Empfehlungen darin, bessere Gesundheitsentscheidungen treffen zu können. Anhand eines Arztprofils werden dem Patienten alle Informationen über die Praxis, von der Adresse bis hin zu den Sprechzeiten, übersichtlich, klar und aktuell bereitgestellt. Der Patient erhält auf einen Blick alle Daten, die er braucht.

*Schnell zum passenden Arzt in Ihrer Nähe durch vielfältige Filtermöglichkeiten*

Und damit die Suche möglichst schnell zum Ziel führt, stehen diverse Filter- und Sortierfunktionen bereit. So kann die Anfrage auf den eigenen Wohnbereich beschränkt und nur nach bestimmten Spezialisten gefiltert werden. Aber auch für Menschen, die sich in der deutschen Sprache wenig sicher fühlen, bietet sich eine Lösung an: Mithilfe einer  Funktion, die Ärzte nach ihren Sprachkenntnissen filtert, kann auch diese Barriere abgebaut werden. Dem notwendigen Gang zum Arzt steht somit nichts mehr im Wege. 

*Profitieren Sie von diesem Servicepaket und erweitern Sie Ihr Angebot.*

Das Arztsuche- Plugin: alle Ärzte auf einem Blick und das völlig kostenfrei. Einfach, schnell und bequem von zu Hause aus! Integrieren Sie das Arztsuche- Plugin einfach und unkompliziert auf Ihrer Homepage und schaffen Sie so einen Mehrwert für Ihre Nutzer.

**Einen API-Key gibts auf [developer.imedo.de](http://developer.imedo.de/ "imedo Arztsuche")**
== Installation ==

1. Laden Sie alle Dateien die Sie mit dem Plugin erhalten haben in wp-content/plugins hoch und aktivieren Sie das Plugin wie gewohnt ueber das WordPress Backend.
1. Beantragen Sie einen API-Key über das Wordpress Backend oder schauen Sie auf **[developer.imedo.de](http://developer.imedo.de/ "imedo Arztsuche")** nach
1. In den Einstellungen vom Plugin (Einstellungen -> Arztsuche) geben sie dann nur noch den API-Key sowie die Groesse der Integration (abhaengig vom API-Key) an.
1. Um die Arztsuche dann auf Ihren Seiten zu integrieren, legen Sie eine neue Seite an, oeffnen den Editor und fuegen den Shortcode [arztsuche] ein. Sollte die Arztsuche dann nicht gerendert werden, fuegen Sie bitte eine Leerzeile vor den Shortcode ein.

== Frequently Asked Questions ==

= Woher bekomme ich einen API-Key? =

Um einen API-Key zu erhalten senden Sie ueber das in den Plugin-Einstellungen zur Verfuegung gestellte Formular eine Anfrage an imedo

= Woher weiss ich die Groesse meiner Integration? =

Die Groesse wurde Ihnen beim zusenden des API-Keys mitgeteilt.
